# AGENTS

This repository is executed by Codex as an engineering agent.

Execution rules:

- Prioritize runnable, testable, and reproducible implementations.
- Prefer synthetic CSI and classical baselines before heavier models.
- Treat DeepMIMO and Sionna as optional dependencies unless explicitly installed.
- Never report fabricated experiment results.
- All plots and tables must be generated by scripts.

Development workflow:

1. Implement the smallest correct feature.
2. Add or update tests.
3. Run smoke checks and `pytest -q`.
4. Commit phase-complete changes.

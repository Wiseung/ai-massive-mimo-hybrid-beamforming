# v0.9.0 Release Notes

## Scope

- Sionna `RZFPrecoder` optional native method bridge
- Sionna `RZFPrecoder` API audit
- adapter bridge from native Sionna output to `PrecoderOutput`
- same-realization semantic validation against `project_rzf`
- quick seed/SNR alignment sweep
- optional method integration into the unified CSI + `PrecoderOutput` demo
- optional Sionna dependency only
- no Sionna RT
- no ray tracing
- no 5G NR full stack

## New Features

- native precoder API audit
- Sionna `RZFPrecoder` adapter bridge
- `sionna_rzf_precoder` optional native method
- same-realization project-vs-Sionna RZF validation
- quick SNR/seed alignment sweep
- updated comparison report
- native precoder artifact manifest
- native precoder minimal reproduction

## Headline Results

- `sionna_rzf_available = true`
- `sionna_rzf_callable = true`
- `converted_to_precoder_output = true`
- `native_receiver_success = true`

- `relationship_status = close_but_different`

- quick sweep keeps callable / converted / native-receiver success on all evaluated rows

## Known Limitations

- not a full native-only benchmark
- `RZFPrecoder` remains an optional native method bridge, not a mainline native replacement
- not strictly equivalent to `project_rzf`
- no Sionna RT
- no ray tracing
- no 5G NR full stack
- not production e2e
- no stable learned `> WMMSE-iter5` claim

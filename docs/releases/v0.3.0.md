# v0.3.0 Release Notes

## Scope

- Sionna OFDM learned beamformer training
- synthetic OFDM channel only
- no Sionna RT
- no ray tracing
- no 5G NR full stack

## New Features

- TinyNeuralBeamformer training baseline
- Residual RZF learned refinement
- Unfolded Lite model
- WMMSE-distilled residual model
- RZF / WMMSE-iter5 evaluation
- multi-seed quick benchmark
- latency / parameter benchmark
- residual correction analysis
- scale sweep quick
- train SNR ablation quick
- teacher leakage audit
- distill weight sweep

## Headline Results

- Tiny is much weaker than the analytic priors:
  `mean_sum_rate = 8.723585`, `gap_to_rzf = -39.5834%`
- Residual-RZF reaches RZF-level performance:
  `mean_sum_rate = 17.657597`, `gap_to_wmmse_iter_5 = -0.4999%`
- Residual WMMSE-distill gives only a noise-level improvement over Residual-RZF:
  `mean_sum_rate = 17.657607`, `gap_to_wmmse_iter_5 = -0.4997%`
- No learned model beats `WMMSE-iter5`.
- Teacher leakage audit passes:
  `teacher_used_during_inference = false`, `leakage_detected = false`
- Residual correction is small, so Residual-RZF should be described as `RZF-level learned refinement`, not an RZF-beating model.

## Known Limitations

- synthetic OFDM only
- quick robustness, not full exhaustive benchmark
- no RT
- no ray tracing
- no 5G NR full stack
- no WMMSE-level learned breakthrough
- not production e2e

## Reproduction Commands

```bash
python scripts/train_sionna_ofdm_beamformer.py \
  --config configs/sionna_ofdm_learned_beamformer.yaml \
  --out outputs/runs/sionna_ofdm_learned_beamformer

python scripts/train_sionna_ofdm_beamformer.py \
  --config configs/sionna_ofdm_residual_rzf.yaml \
  --out outputs/runs/sionna_ofdm_residual_rzf

python scripts/train_sionna_ofdm_beamformer.py \
  --config configs/sionna_ofdm_unfolded_lite.yaml \
  --out outputs/runs/sionna_ofdm_unfolded_lite

python scripts/train_sionna_ofdm_beamformer.py \
  --config configs/sionna_ofdm_residual_wmmse_distill.yaml \
  --out outputs/runs/sionna_ofdm_residual_wmmse_distill

python scripts/compare_sionna_ofdm_training_runs.py \
  --tiny outputs/comparisons/sionna_ofdm_learned_beamformer \
  --residual outputs/comparisons/sionna_ofdm_residual_rzf \
  --unfolded outputs/comparisons/sionna_ofdm_unfolded_lite \
  --wmmse-distill outputs/comparisons/sionna_ofdm_residual_wmmse_distill \
  --out outputs/comparisons/sionna_ofdm_training_family_v2

python scripts/audit_sionna_teacher_leakage.py \
  --config configs/sionna_ofdm_residual_wmmse_distill.yaml \
  --ckpt outputs/runs/sionna_ofdm_residual_wmmse_distill/best.pt \
  --out outputs/comparisons/sionna_ofdm_teacher_leakage_audit

python scripts/generate_sionna_training_artifact_manifest.py \
  --out outputs/sionna_training_artifact_manifest.json
```

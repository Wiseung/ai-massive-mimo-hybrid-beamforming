# v0.2.0 Release Notes

## Scope

- optional Sionna PHY/OFDM demos
- `sionna-no-rt`
- no Sionna RT
- no ray tracing
- no 5G NR full stack

## New Features

- Sionna environment check
- Sionna API introspection
- Sionna PHY AWGN demo
- Sionna OFDM API introspection
- Sionna OFDM ResourceGrid demo
- Sionna OFDM beamforming bridge
- differentiable learned beamformer
- beamformer gradient check
- few-step OFDM differentiable optimization demo

## Headline Smoke Results

- PHY AWGN demo:
  - `mse = 0.0999`
  - `ber = 7.82e-4`
  - real `sionna.phy.channel.AWGN` used
- OFDM ResourceGrid demo:
  - `mse = 0.1032`
  - `ber = 0.001953`
  - real `ResourceGrid/Mapper/Demapper/RemoveNulledSubcarriers/AWGN` used
- OFDM beamforming bridge:
  - `rzf_ofdm_average_mse = 0.1017`
  - `wmmse_iter_5_ofdm_average_mse = 0.0977`
  - `rzf_ofdm_average_sum_rate = 19.9374`
  - `wmmse_iter_5_ofdm_average_sum_rate = 20.0129`
- Differentiable OFDM beamforming demo:
  - `initial_loss = 2.0943`
  - `final_loss = 0.3303`
  - `loss_decreased = true`
  - `grad_norm = 1.0686`

## Limitations

- no RT
- no ray tracing
- no full Sionna end-to-end training
- no production training pipeline
- toy learned beamformer only

# v0.1.0 Release Notes

## Scope

- synthetic narrowband MU-MISO benchmark
- DeepMIMO `asu_campus_3p5` filtered benchmark
- `K=4`, `Nt=8`, `Nsc=1`

## Main Features

- analytic baselines: `MRT`, `ZF`, `RZF`, `WMMSE`, `WMMSE-iter-k`
- learned models: `CNN`, residual refinements, unfolded `WMMSE-lite`
- unified SE-vs-SNR evaluation and unified latency protocol
- DeepMIMO random-split and contiguous-split benchmark exports
- artifact manifest for benchmark outputs
- short minimal reproduction smoke script
- GitHub Actions CI for `compileall` and `pytest`

## Headline Results

- `WMMSE` is the strongest tested full synthetic reference.
- `wmmse_iter_5` is the strongest reduced-iteration structured reference in the current release.
- The best current `WMMSE-lite` variant matches `wmmse_iter_5` in SE but not in latency.
- The dominant `WMMSE-lite` latency hotspot is the WMMSE initializer, not the learned refinement layers.

## Known Limitations

- no `Nt=64` DeepMIMO result is claimed
- no wideband benchmark is claimed
- no Sionna end-to-end benchmark is claimed
- no claim is made that `WMMSE-lite` is lower-latency than `wmmse_iter_5`

## Reproduction Commands

```bash
python scripts/reproduce_minimal.py \
  --out outputs/repro/minimal_repro_summary.json
```

```bash
python scripts/benchmark_latency.py \
  --data outputs/data/synthetic_narrowband.pt \
  --methods mrt rzf wmmse_iter_1 wmmse_iter_2 wmmse_iter_5 wmmse cnn residual_rzf unfolded_rzf unfolded_wmmse_lite \
  --batch-size 512 \
  --warmup-runs 20 \
  --timed-runs 100 \
  --out outputs/comparisons/latency_v2
```

```bash
python scripts/run_deepmimo_model_family_benchmark.py \
  --data outputs/data/deepmimo_asu_campus_3p5_narrowband.pt \
  --seeds 1 2 3 \
  --split-mode random \
  --methods rzf wmmse_iter_5 cnn residual_rzf unfolded_wmmse_lite \
  --out outputs/comparisons/deepmimo_model_family_random
```

```bash
python scripts/run_deepmimo_model_family_benchmark.py \
  --data outputs/data/deepmimo_asu_campus_3p5_narrowband.pt \
  --seeds 1 2 3 \
  --split-mode contiguous \
  --methods rzf wmmse_iter_5 cnn residual_rzf unfolded_wmmse_lite \
  --out outputs/comparisons/deepmimo_model_family_contiguous
```

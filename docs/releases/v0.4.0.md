# v0.4.0 Release Notes

## Scope

- Sionna-native OFDM receiver chain
- analytic and learned beamformer insertion
- synthetic / project-H_f-assisted
- optional Sionna dependency only
- no Sionna RT
- no ray tracing
- no 5G NR full stack

## New Features

- native OFDM baseline chain
- pilot pattern audit
- estimator/equalizer minimal demo
- beamformed receiver shape bridge
- project `RZF` / `WMMSE-iter` insertion
- learned `residual_rzf` insertion
- learned `residual_wmmse_distill` insertion
- SNR mini benchmark
- no teacher leakage during learned inference
- native-chain artifact manifest
- minimal native-chain reproduction command

## Headline Results

- The native receiver path now succeeds for both analytic and learned methods through real Sionna `channel` / `estimator` / `equalizer` / `demapper` components.
- `learned_residual_rzf` reaches native receiver success with `teacher_used_during_inference = false`.
- `learned_residual_wmmse_distill` also reaches native receiver success with `teacher_used_during_inference = false`.
- The learned methods remain close to the analytic baselines inside the current synthetic/project-H_f-assisted benchmark.
- There is no stable claim that a learned method universally exceeds `WMMSE-iter5`.

## Known Limitations

- precoder / `H_f` side remains project-assisted
- not a full native-only benchmark
- synthetic setting only
- no Sionna RT
- no ray tracing
- no 5G NR full stack
- not production e2e
- mini benchmark only

## Reproduction Commands

```bash
python scripts/sionna_native_ofdm_baseline_chain.py \
  --out outputs/sionna_native_chain/baseline_chain_summary.json

python scripts/sionna_native_ofdm_beamforming_chain.py \
  --out outputs/sionna_native_chain/beamforming_receiver_chain_v2_summary.json \
  --enable-receiver-chain \
  --receiver-mode auto

python scripts/sionna_native_ofdm_learned_beamforming_chain.py \
  --out outputs/sionna_native_chain/learned_beamforming_receiver_summary.json \
  --receiver-mode auto

python scripts/compare_sionna_native_learned_beamforming.py \
  --analytic-summary outputs/sionna_native_chain/beamforming_receiver_chain_v2_summary.json \
  --analytic-metrics outputs/sionna_native_chain/beamforming_receiver_chain_v2_metrics.csv \
  --learned-summary outputs/sionna_native_chain/learned_beamforming_receiver_summary.json \
  --learned-metrics outputs/sionna_native_chain/learned_beamforming_receiver_metrics.csv \
  --out outputs/sionna_native_chain

python scripts/run_sionna_native_learned_chain_minibench.py \
  --out outputs/sionna_native_chain/native_learned_minibench

python scripts/generate_sionna_native_chain_artifact_manifest.py \
  --out outputs/sionna_native_chain/native_chain_artifact_manifest.json
```

# v1.0.0-rc1 Release Notes

## Scope

- interface-first Sionna bridge release candidate
- native channel extraction
- `ExtractedCSI` interface
- CSI consumer unification
- `PrecoderOutput` interface
- Sionna `RZFPrecoder` optional native method bridge
- native precoder contract hardening
- optional Sionna dependency only
- no Sionna RT
- no ray tracing
- no 5G NR full stack

## Release Lineage

- `v0.1.0` benchmark prototype
- `v0.2.0` optional Sionna PHY/OFDM demos
- `v0.3.0` learned beamformer training
- `v0.4.0` native receiver chain
- `v0.5.0` channel extraction bridge
- `v0.6.0` CSI interface + same-batch equivalence
- `v0.7.0` CSI consumer unification
- `v0.8.0` `ExtractedCSI + PrecoderOutput` bridge
- `v0.9.0` Sionna `RZFPrecoder` optional method bridge
- `v1.0.0-rc1` contract-hardened interface candidate

## Headline Results

- `ExtractedCSI` created from a Sionna channel tensor
- `PrecoderOutput` created by analytic / learned / optional Sionna RZF paths
- Sionna `RZFPrecoder` callable and convertible
- native receiver path success
- same-batch equivalence checks passed where applicable
- contract validation passed
- regression matrix passed
- `aliasing_project_rzf_detected = false`

## Known Limitations

- not a full native-only benchmark
- Sionna `RZFPrecoder` remains an optional native method bridge, not a mainline replacement
- `relationship_status = close_but_different`
- strict `project_rzf` equivalence is not claimed
- no Sionna RT
- no ray tracing
- no 5G NR full stack
- not production e2e
- no stable learned `> WMMSE-iter5` claim
